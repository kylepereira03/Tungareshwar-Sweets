# 🍬 Tungareshwar - Sweets & Namkeen Website

This is the frontend source code for **Tungareshwar**, a website created for a local sweets shop as a part of my final year project in the Diploma in Computer Engineering program.

## 🌐 Project Overview

**Tungareshwar** is a responsive and visually appealing website designed to showcase a variety of traditional and modern sweets. The goal of this project was to build a user-friendly website that highlights the shop's offerings, encourages customer engagement, and provides essential information like shop location, contact details, and product highlights.

> **Note:** This repository contains only the frontend part of the website (HTML, CSS, JavaScript). There is no backend or database integration included.

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
